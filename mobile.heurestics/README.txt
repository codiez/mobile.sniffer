This package contains heurestics (rules-of-thumb) when dealing with mobile phones and web systems.

Hearestics are divided to two categories

* mobile.heurestics.useragent: User agent based rules. Need mobile.sniffer package support.

* mobile.heurestics.simple: Rules which can be applied based on plain HTTP headers.

Rules covered
-------------

* Extracting user agent information from HTTP headers

* Detecting high-end and low-end mobile phones

* Formatting phone number links (iPhone / WTAI)
